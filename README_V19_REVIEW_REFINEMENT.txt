Version 19 — Auto Sort Review refinement

- Changes now lists only points whose visible Side/Segment + number actually changed.
- Internal manualSeq updates no longer create false entries such as E3 → E3.
- Clicking a review card never scrolls, recentres, or changes zoom.
- Clicking only flashes the point briefly in its current location.
- Unchanged points receive no review marker.
- True number changes use a restrained blue square highlight.
- Warnings use amber; high-attention warnings use red.
- Review marks remain temporary and are never exported.
